My Independent Study Project Plan

IDENTITY

A bullet hell game where you are the only one that cant shoot bullets. You can only deflect bullets or swing in a melee range. The player will have a few actions dodge, roll, parry, swing. There will also be a timing aspet with the swing. Where if you swing at the right time you can do double damage. The world will be procedurally generated for a different experience each time. There will be many diffent types of levels that will have there own characteristics to make them feel different. I kind of want the game to have a pretty hard curve.

PROBLEM

Bullet Hells are getting to easy, people need a challenge. I wanted to create a game that is not hard to get into but has a high skill ceiling.

SOLUTION

My solution is to create a game that is a different type of bullet hell.

TARGET MARKET

My targeet audience is the Rougelike bullet hell audience.

COMPETITION

My competition would be other games of the same type. I think we do something different to the point where it will entice people to play.

REVENUE STREAMS

My revenu stream would be of game sells. If not game sells maybe in-app purchaes or adds.

MARKETING STREAMS

I can market this project through differnt gamming communites or paying youtubers or twitch streamers to try out the game.

EXPENSES

My expences would be my time and development. And If i want to truly sell this to people I need to get a artist to design the art for the game.

TEAM AND KEY ROLES

I dont expect there to be a big team for this so for an team. We would have A Project Lead that would oversee eveything. Then a person that would see the art direction and where that would go.
